# 🌐 Prática HTML - Tabelas, Listas e Formulário

## 📌 Descrição
Projeto de prática em **HTML e CSS**, com foco em construção de tabelas, listas e formulários.  
O objetivo é treinar a estruturação de páginas web e aplicar estilos básicos com CSS.

---

## 🚀 Etapas principais
1. Criação de tabelas simples e estilizadas com CSS.
2. Implementação de listas ordenadas e não ordenadas.
3. Desenvolvimento de formulários com campos de formulário.
4. Organização dos arquivos HTML para prática e revisão.

---

## 🛠️ Tecnologias
- HTML5  
- CSS3  

---

## 📊 Resultado
Páginas web simples e funcionais, contendo tabelas, listas e formulários estilizados para prática de fundamentos de front-end.
